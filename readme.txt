WF Weather allows the user to integrate weather information provided by various providers.

== Description ==

WF Weather is the `wunderfarm-way` to integrate weather information provided by various providers in a truly responsive box. Very easy, 100% responsive, fast and SEO-friendly!

It supports the integration of South Tyrolean weather information provided by provinz.bz.it/wetter (7 districts).

== Installation ==
1. Upload `WF Weather` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If you want to set custom settings, go to Settings menu > WF Weather.
